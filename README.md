# Tic-Tac-Toe ❌⭕

Tic-Tac-Toe Game 🎮
Классическая игра "Крестики-нолики" с графическим интерфейсом на JavaFX и встроенным AI-противником.

📋 Описание
Это полнофункциональная реализация игры Tic-Tac-Toe (Крестики-нолики) с двумя режимами игры:

Игрок против Игрока - классический режим для двух игроков
Игрок против AI - играйте против умного компьютерного противника
Игра проверяет:

* победу по строке
* победу по колонке
* победу по диагонали
* ничью

---

## 🧠 Правила игры

1. Игра проходит на поле **3×3**.
2. Первый игрок ставит **X**, второй — **O**.
3. Игроки делают ход по очереди.
4. Побеждает тот, кто первым соберёт **3 символа подряд**:

   * по горизонтали
   * по вертикали
   * по диагонали
5. Если поле заполнено и победителя нет — объявляется **ничья**.

## 🚀 Как запустить

1. Склонируйте репозиторий:

```bash
git clone https://github.com/username/tic-tac-toe.git
```

2. Перейдите в папку проекта:

```bash
cd tic-tac-toe
```

3. Запустите программу.

Пример для Python:

```bash
python main.py
```
