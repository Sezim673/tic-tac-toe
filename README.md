# 🎮 Tic-Tac-Toe

Небольшая реализация классической игры **Крестики-нолики** с простым интерфейсом на HTML, CSS и JavaScript.

---

## 📋 Описание

Это полнофункциональная реализация игры Tic-Tac-Toe (Крестики-нолики) с двумя режимами игры:

1. Игрок против Игрока - классический режим для двух игроков

2. Игрок против AI - играйте против умного компьютерного противника

---

## ✨ Особенности

* 🎯 Классическая логика игры
* 🎨 Простой и понятный интерфейс
* 🔄 Возможность начать новую игру
* 🏆 Автоматическое определение победителя

## 🎮 Как играть

1. Игроки по очереди ставят **X** и **O**
2. Побеждает тот, кто первым соберёт **3 символа подряд**
3. Если поле заполнено и победителя нет — объявляется **ничья**

---

## 🤖 Стратегия AI
AI-противник использует следующую приоритетную стратегию:

Победа - если есть возможность выиграть, делает победный ход
Блокировка - блокирует победный ход игрока
Центр - занимает центральную клетку, если свободна
Углы - предпочитает угловые позиции
Любая клетка - занимает любую свободную клетку
