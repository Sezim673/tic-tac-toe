# 🎮 Tic-Tac-Toe

Простая реализация игры **Крестики-нолики** с графическим интерфейсом на JavaFX и возможностью играть против компьютера.

---

## 📋 Описание

Это небольшое приложение, в котором реализована классическая игра **Tic-Tac-Toe**.
Игрок может играть либо против другого игрока, либо против компьютерного противника.

---

## ✨ Возможности

* 🎨 Графический интерфейс на **JavaFX**
* 👥 Режим **игрок против игрока**
* 🤖 Режим **игрок против AI**
* 🔄 Возможность начать новую игру
* 🏆 Автоматическое определение победителя или ничьей

---

## 🏗️ Структура проекта

```
src/
 ├── GameModel.java
 ├── GameView.java
 ├── HelloController.java
 ├── AIPlayer.java
 └── HelloApplication.java
```

* **GameModel** — хранит состояние игры и проверяет победу
* **GameView** — отвечает за интерфейс
* **HelloController** — обрабатывает действия пользователя
* **AIPlayer** — реализует логику компьютерного игрока

---

## 🚀 Запуск

Требования:

* Java 11+
* JavaFX

Запустите файл **HelloApplication.java** через IDE.

---

## 🎮 Как играть

1. Запустите приложение
2. Выберите режим игры
3. Нажимайте на клетки поля, чтобы сделать ход
4. Игрок **X** ходит первым
